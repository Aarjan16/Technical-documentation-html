Introduction to PHP

PHP (Hypertext Preprocessor) is a popular open-source server-side scripting language designed primarily for web development but also used as a general-purpose programming language. PHP scripts are executed on the server, and the result is returned as plain HTML to the client's browser. PHP can interact with databases, handle forms, manage sessions, and perform many server-side tasks dynamically.

Palindrome or not


<?php
function isPalindrome($string) {
    $cleaned = strtolower(str_replace(' ', '', $string));
    $reversed = strrev($cleaned);
    return $cleaned === $reversed;
}

$input = "Madam";

if (isPalindrome($input)) {
    echo "'$input' is a palindrome.";
} else {
    echo "'$input' is not a palindrome.";
}
?>
